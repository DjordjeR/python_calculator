# Python Calulator
Simple command line calculator built in Python 3 using eval function ( not really redommended )
```
eval(equation)
```

## Usage
Type 'quit to close the script

Type your equation is the following form:
```
5 + 6

7*8
```
## Examples
```
Enter equation:5 * 7

35+8

43
```
